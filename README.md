# FitStart

Мобильное приложение фитнес-клуба **FitStart** (Flutter, только верстка).

## Структура

```
lib/
├── main.dart                      # точка входа + маршруты
├── theme.dart                     # цветовая палитра и тема
├── widgets/                       # переиспользуемые виджеты
│   ├── curved_header.dart         # изогнутая шапка (авторизация/регистрация)
│   ├── labeled_field.dart         # поле ввода с подписью
│   └── placeholder_box.dart       # заглушка вместо фото/видео
└── screens/
    ├── onboarding.dart            # 01 — Онбординг (с логотипом)
    ├── auth.dart                  # 02 — Авторизация
    ├── registration.dart          # 03 — Регистрация (+ Отчество, Дата рождения)
    ├── home.dart                  # контейнер с нижней навигацией
    ├── action_page.dart           # 12 — Action (Тренировки)
    ├── subscription.dart          # 14 — Подписка
    ├── settings.dart              # 15 — Настройки
    ├── qr_pass.dart               # 16 — QR Пропуск
    ├── bottom/                    # вкладки нижней навигации
    │   ├── bottom_home.dart       # 04 — Главная
    │   ├── bottom_schedule.dart   # 05 — Расписание
    │   ├── bottom_trainers.dart   # 06 — Тренеры
    │   ├── bottom_gym.dart        # Зал (Тренажёры + Упражнения)
    │   └── bottom_profile.dart    # 13 — Профиль
    ├── gym/
    │   ├── equipment_tab.dart     # 08 — Тренажёры
    │   └── exercises_tab.dart     # 10 — Упражнения
    └── details/                   # всплывающие панели
        ├── trainer_details.dart   # 07 — Тренер (подробно)
        ├── equipment_details.dart # 09 — Тренажёр (детали)
        └── exercise_details.dart  # 11 — Упражнение (детали)
```

## Навигация

- Онбординг → «Начать» → Регистрация; «Войти» → Авторизация
- Авторизация / Регистрация → Главная (нижняя навигация)
- Нижняя навигация: Главная · Расписание · Тренеры · Зал · Профиль
- Главная → «Пропуск в клуб» → QR Пропуск; «Расписание Все ›» → Action (Тренировки)
- Тренеры / Тренажёры / Упражнения → тап по карточке → всплывающая панель с деталями
- Профиль → карточка подписки → Подписка; шестерёнка → Настройки
- Настройки → «Выйти» → Онбординг

> Только верстка. Логика подключения к БД не реализована.
